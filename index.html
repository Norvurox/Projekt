    Willkommen auf unserer Website, die sich über das Spiel Counter Strike: Global Offensive dreht. Im folgenden sehen Sie eine kurze Erklärung und einen Trailer, welches dieses Spiel vorstellt. Counter Strike: Global Offensive ist ein Online-Taktik-Shooter, welches von Valve entwickelt wurde und am 21. August 2012 erschienen ist. Es ist das vierte Spiel der Counter Strike Reihe und der Nachfolger von Counter Strike Source. Es ist für mehrere Plattform erhältlich, unter anderem für Windows, Mac, Linux, aber auch für bestimmte Konsolen. Das Spiel läuft über die Source-Engine und kann schon mit niedrigen Systemvoraussetzungen gespielt werden.
